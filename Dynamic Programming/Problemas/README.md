# Problemas

 ### Subset Sum  
   #### Verificar si es posible obtener un número A a base de los números dentro de un arreglo dado.  
   
 ### Digit DP    
   #### Contar cuántos números entre A y B, donde A y B son números naturales, cumplen con condiciones determinadas.
